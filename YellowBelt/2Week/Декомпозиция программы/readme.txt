в оригинальном варианте компановки все методы константны и принимают константные переменные, что при добавлении нового маршрута вызывала переполнение переменной типа Query в векторе stops. Вызвано это тем, что она определена до цикла в функции main. Для решения этой проблемы метод Add был лишен константности в аргументе вектора. В авторском решении добавлена возможно изменения размера до 0 в операторе присвоения.
